# Kairove Project Index

## 0. One-Line Definition

**影潮枢 / Kairove** is a personal AI video production system that discovers trending video formats, extracts reusable format structures, semantically transfers them into new contexts, generates videos, checks quality, retries, packages, publishes, and learns from feedback.

It is not a simple prompt generator and not an MMD-only project.

## 6. Non-Negotiable Principles

1. Kairove is not MMD-only.
2. MMD/3D is one specialized route, not the main identity.
3. First production line is ordinary AI format reproduction.
4. Source, asset, decision, quality, retry, and publish steps must be traceable.
5. Research assets, local assets, and generated assets stay separate.
6. Official assets can be direct-use under current policy.
7. Personal creator and unknown assets require review before direct use.
8. Permissions are capability switches, not linear levels.
9. Score weights must be visible, configurable, versioned, and recalculable.
10. Learning suggestions require approval before becoming rules.
11. Do not fake external APIs.
12. Do not auto-publish unless permissions and platform setup allow it.
13. Detailed phase drafts are not implementation approval by themselves.
